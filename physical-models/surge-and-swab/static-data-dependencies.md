# Static Data Dependencies

## **Dependencies - Static Data**

* From Well Parameter (Planning) or Interventions (Real-time):
  *   Trajectory

      * Measured Depth
      * Inclination&#x20;
      * Azimuth


  *   Geopressures (for max  running speed)

      * Pore Pressure
      * Fracture Pressure


* From Interventions -> Sections -> Run:
  * Fluids&#x20;
    * Density
    * Rheological model
  * Wellbore&#x20;
    * Inner Diameter
    * Start Depth
    * End Depth
  * Drill String&#x20;
    * Outer Diameter
    * Inner Diameter
    * Length
  * BHA&#x20;
    * Outer Diameter
    * Inner Diameter
    * Length
